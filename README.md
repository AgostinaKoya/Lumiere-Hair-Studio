## Descripción

Aplicación web para la gestión de turnos de peluquería. Los usuarios pueden reservar un turno paso a paso — eligiendo el servicio, la fecha y el horario disponible — y consultar todas sus reservas en la sección `MisTurnos`.

---

## ✨ Funcionalidades

- ▸ Reserva de turnos paso a paso: servicio → fecha → horario
- ▸ Sección `MisTurnos` para ver y gestionar reservas
- ▸ Autenticación segura con JWT
